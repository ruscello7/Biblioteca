# Library Management System

A Library management system built in Java with layered architecture (Model → Repository → Service), implementing OOP concepts and CRUD operations.

## Features
- Register and list books
- Register users
- Create loans with availability check
- Return books
- List open loans
- Input validation and error handling

## Technologies
- Java
- OOP (Encapsulation, Composition, Dependency Injection)
- ArrayList (in-memory storage)
- IntelliJ IDEA
- Git & GitHub

## Architecture
src/
model/ → Book,User,Loan
repository/ → BookRepository, UserRepository, LoanRepository
service/ → BookService, UserService, LoanService
Main.java → Interactive menu with Scanner 

## How to run
1. Clone the repository: git clone: https://github.com/ruscello7/Biblioteca.git`
2. Open in IntelliJ IDEA
3. Run `Main.java`
